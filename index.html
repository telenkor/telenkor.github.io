Migrator

Fast save or restore macOS applications and their settings

View on GitHub

About

Migrator is useful if you are reinstalling macOS or want to transfer configured applications to another computer. You don’t have to open .dmg files of each application, drag each one into Applications or run installers and re-enter a huge number of settings.

Screenshots

Key Features

Security

Migrator itself does not scan your Applications directory. You enter the data into it yourself.

Nothing extra

There is no need to collect any service databases, caches, or large temporary files.

Native

Built with AppleScript, ensuring smooth integration without requiring third-party software.

Quick Start

Download the latest version from the Releases page and move it to your Applications folder.
