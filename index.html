Migrator

Efficient CLI tool for media library migration and organization

View on GitHub

About

Migrator is a powerful command-line utility designed to help users move, organize, and back up large media collections. Whether you are migrating photos from an old hard drive or restructuring your home server, Migrator ensures your files are handled safely and efficiently.

Key Features

Smart Filtering

Automatically identifies file types and prevents duplicates during the migration process.

Structure Preservation

Maintain your original directory tree or reorganize files based on metadata like dates.

High Performance

Optimized for speed when dealing with thousands of high-resolution images and videos.

Quick Start

To start using Migrator, clone the repository and install the requirements:

git clone https://github.com/telenkor/migrator.git
cd migrator
pip install -r requirements.txt

Basic usage example:

python main.py --source /path/to/source --dest /path/to/destination
